How to fix Atheros A8161 / Atheros Killer E2200 Ethernet in Linux

2015-05-30 :: tutorials, linux

I just spent a significant number of hours trying to fix the Ethernet card in my MSI GS60 laptop on Ubuntu 15.04. Turns out there’s a bug in the drivers that’s been around since February (!), and it’s still present on version 4.0.4 of the kernel. The workaround is to set the MTU to 8192. This works for both the A8161 and the E2200 card. Assuming you’re on eth0,

sudo ifconfig eth0 mtu 8192
